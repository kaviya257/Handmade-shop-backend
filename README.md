🧶 Handmade Shop - Backend System
A full-stack e-commerce backend system for a Handmade Products Marketplace built using Java, Spring Boot, MySQL, HTML, and CSS.
This project simulates a real-world shopping platform with role-based access (Admin & User), product management, cart system, and order processing.

🚀 Project Overview
The Handmade Shop System is designed to provide a seamless digital marketplace for handmade goods.
It supports secure authentication, structured product management, and complete order lifecycle handling.
This project follows MVC architecture and demonstrates backend development best practices using Spring Boot.

✨ Key Features

👤 User Module
User registration and login
Browse handmade products
Add products to cart
Place orders
View order history

🛠 Admin Module
Add / update / delete products
Manage inventory
View all customer orders
Monitor platform activity

🧾 Order System
Cart-based purchasing workflow
Order placement and confirmation
Order tracking system
Order history per user
